# Release Notes - v0.8.0

## Features
- **Elegant Ticker Entry**: Completely redesigned the "Add New Widget" experience with a modern, borderless prompt window.
- **Popular Ticker Lookup**: Added a new "Browse" screen that allows you to easily find and add top Stocks, ETFs, and Cryptocurrencies.
- **Clickable Suggestions**: The ticker entry window now features one-tap clickable suggestions for common indices and assets.
- **Contextual Actions**: Added "Add New Widget", "Release Notes", and "Check for Updates" to the right-click menu of all active widgets.
- **Update Notifications**: Implemented a built-in update checker that notifies you when a newer version is available on GitHub and provides a direct download link.
- **Integrated Documentation**: Quick access to version-specific release notes directly from the app.

## Improvements
- **Default Preferences**: Changed the default for "Keep on Top" to unchecked, providing a smoother experience for new widgets.
- **Draggable Entry Window**: The new ticker input and lookup windows are fully movable by clicking and dragging their backgrounds.
- **Expanded Layout**: Increased the default widget height to better accommodate diverse stock information.

## Fixes
- **X-Axis Label Stabilization**: Implemented dynamic scroll-correction for the Classic site view. This prevents time-axis labels from being cut off at the bottom of the widget when extra pricing data (like After Hours or Post-market quotes) is present.
- **Layout Robustness**: Fixed an issue where the ticker input box could become obscured or collapsed in certain window states.
